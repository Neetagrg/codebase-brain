# Codebase Brain 🧠

Every developer knows this pain: you spend 30 messages 
explaining your codebase to an AI, get a partial answer, 
then start a new session and explain everything again from 
scratch. Forever.

**Codebase Brain solves this.**

It uses IBM Bob to generate a permanent "brain" for any 
complex codebase — so any developer, or any future Bob 
session, has full context instantly. No more starting from 
zero. No more re-explaining. Just answers.

---

## What It Does

**AGENTS.md** — Bob reads your entire repo and generates a 
precise technical document capturing every file's purpose, 
every non-obvious pattern, every failure mode, and the full 
architecture. One document. Permanent context.

**Slash Commands** — Custom commands like `/debug-joint` and 
`/lua-trace` that encode domain expertise. Any developer 
invokes one command and Bob immediately knows what to do.

**Domain Skill** — A reusable Bob skill that turns Bob into 
a senior engineer on your specific codebase the moment it's 
activated.
